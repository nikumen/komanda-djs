# КОМАНДА DJS

Лендинг диджей-интенсива «КОМАНДЫ» (месяц работы с тремя кураторами: Никита Меньшов, Никита Соболь, Сергей Донских).

Vanilla HTML/CSS/JS + GSAP + three.js. 3D hero-колонка (Funktion-One) и посекционные 3D-акценты (микшер DJM-A9, пэд-контроллер, наушники AIAIAI, винил Locked Club).

Live: https://nikumen.github.io/komanda-djs/
